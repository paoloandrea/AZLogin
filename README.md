AZLogin
=======

The best way to implement a login in your iOS application

-----------------------------------------------------------

1. Copy all file in "Login" folder to your project

2. Download on github this library:
  - SBTableAlert
  - AFNetworking
  - TTTAttributedLabel
  - UIGlossyButton
  - TPKeyboardAvoiding
  - SVProgressHUD

3. Check the AZConstans.h for configuration vars
